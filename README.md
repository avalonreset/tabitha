## What is Tabitha?
Tabitha is a community fork of **Tabby** (MIT‑licensed) focused on quality‑of‑life improvements and a cleaner, Windows‑friendly workflow. It keeps the power of the original project while refining input, clipboard behavior, and day‑to‑day usability.

## Highlights
- Smart clipboard paste: handles **text and images** without fighting you
- Great Windows ergonomics (Win+Shift+S screenshots, Win+H dictation, native shortcuts)
- Updated Clink for a better Windows shell experience
- Tabs, panes, profiles, and session management
- SSH, SFTP, Telnet, and Serial support
- Cross‑platform (Windows, macOS, Linux)
